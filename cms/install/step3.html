03. 填写初始化配置信息 数据库及相关初始信息配置

填写数据库服务器信息

  • 数据库地址： 本机请填写localhost
  • 数据库端口： 一般情况下是3306
  • 数据库名： 用于安装的数据库名称，若不存在将尝试创建
  • 数据库用户名： 用于数据库连接验证的用户名
  • 数据库密码： 用于数据库连接验证的密码
  • 数据库表前缀： 在原有数据库运行BookCms时，请修改前缀
  • 强制安装： 是　 否 强制安装将会删除原有的表

填写站点初始化信息

  • 管理员账号： 初始管理员账号
  • 管理员Email： 请输入管理员Email
  • 管理员密码： 请输入管理员密码
  • 重复管理员密码： 重复输入上面的管理员密码
